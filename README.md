# CatchFade

CatchFade is a Python-based project for collecting sensor data, detecting anomalies, generating briefings, and serving a dashboard.

## Project structure

- `src/` – core data collection, anomaly detection, and briefing logic
- `dashboard/` – dashboard application
- `docs/` – documentation generation scripts

## Getting started

1. Create a virtual environment.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application components as needed from `src/` and `dashboard/`.
